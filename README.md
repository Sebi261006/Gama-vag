# VAG Showroom

Un site showroom interactiv pentru gama completă Volkswagen Audi Group.

## Brandurile incluse:
- **Volkswagen** - Compacte și electrice accesibile
- **Audi** - Sedane și SUV-uri premium
- **Porsche** - Supercaruri și vehicule de performance
- **Lamborghini** - Hypercaruri și supercaruri extreme
- **Bentley** - Vehicule ultra-lux britanic
- **Skoda** - Valoare și practicitate
- **Cupra** - Spirit sportiv spaniol

## Design:
- Layout modern cu glassmorphism
- Carduri interactive cu imagini de înaltă calitate
- Responsive și elegant
- Animații smooth și hover effects

## Fișiere:
- `index.html` - Pagina principală cu toate brandurile
- `volkswagen.html`, `audi.html`, `porsche.html`, etc. - Pagini cu modele specifice
- `style.css` - Stiluri moderne și responsive
